# Peniel's Kitchen - Restaurant Website

A responsive, high-converting restaurant website built with Bootstrap 5, HTML, CSS, and JavaScript.

## Features

### Core Features
- Responsive design (mobile-first approach)
- Bootstrap 5 grid system and components
- Sticky navbar that shrinks on scroll
- Hero section with image slider
- Interactive menu with category filtering
- Gallery with lightbox modal
- Customer testimonials section
- Location and contact information
- Promotional banner with countdown timer
- Dark mode toggle
- Smooth scroll animations
- Back-to-top button
- WhatsApp ordering integration

### Technical Features
- SEO-friendly HTML structure
- Open Graph meta tags for social sharing
- Accessible design (ARIA labels, keyboard navigation)
- Lazy loading for images
- Modular CSS with CSS custom properties
- Clean, organized JavaScript
